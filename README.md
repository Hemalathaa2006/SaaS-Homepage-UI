# Flowzy SaaS Homepage

Flowzy SaaS Homepage is a clean and modern landing page designed for a software-as-a-service platform.  
This project is created using **HTML and Tailwind CSS**.

## Features

- Professional SaaS-style homepage layout
- Navigation bar with brand name
- Hero section with call-to-action
- Features section
- Pricing section
- Clean footer design
- Responsive layout

## Technologies Used

- HTML
- Tailwind CSS
